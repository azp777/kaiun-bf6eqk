# ✦ 開運コンパス ✦

七つの占術が導く、心の道しるべ。生年月日を入れるだけで7種類の占術をまとめて診断できる総合占いWebアプリです。

## 機能

- **数秘術** — ライフパスナンバー(マスターナンバー対応)・バースデーナンバー・パーソナルイヤー
- **吉方位(九星気学)** — 本命星、年盤・月盤のSVGコンパス、五黄殺・暗剣殺・本命殺・本命的殺・歳破/月破の判定、祐気取りガイド
- **西洋占星術** — 太陽星座・エレメント・守護星
- **マヤ暦** — ドリームスペル準拠のKIN・太陽の紋章・銀河の音・ウェイブスペル
- **六星占術(簡易版)** — 運命星と12年運気カレンダー(大殺界表示)
- **神社属性(繭気属性)** — 生年月日+血液型による属性と相性の良い聖地タイプ
- **16タイプ性格診断** — 12問の簡易テスト付き
- **💞 相性診断** — 最大7占術で二人の縁を鑑定(実アスペクト角・五行相生相剋・神秘KIN等の本格判定)
- **🔮 開運相談室** — 診断結果と連動するオフライン自動応答チャット
- **今日の運勢** — 誕生日×日付で毎日変わる5項目の運勢

## 使い方

`index.html` をブラウザで開くだけ。データはすべて端末内(localStorage)に保存され、外部への通信は一切行いません。

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | 画面・スタイル |
| `uranai-data.js` | 占術データ・計算ロジック |
| `uranai-main.js` | 描画・画面制御 |
